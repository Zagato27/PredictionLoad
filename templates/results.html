Описание:

  • Средняя задержка — оценка среднего времени отклика при целевом RPS по выбранной модели.
  • Максимальная — оценка пикового времени: средняя × коэффициент (max/avg базовой ступени).

Подробнее по моделям:

  • M/M/1 (один сервер): одна очередь и один канал обслуживания; вход — пуассоновский поток, время сервиса — экспоненциальное. Нагрузка ρ = X·S. R = S / (1 - ρ) при ρ < 1; при ρ ≥ 1 — «∞».
  • M/M/c (несколько серверов): одна очередь и c параллельных каналов. R = 1/μ + C(c,a)/(cμ - λ).
  • G/G/1 (Кингман): R ≈ S + (ρ/(1-ρ)) · ((C_a^2 + C_s^2)/2) · S — учитывает вариативность.
  • G/G/c (Allen–Cunneen): R ≈ S + ((C_a^2 + C_s^2)/2) · C(c,a)/(cμ - λ) — вариативность при c каналах.

Как читать столбцы:

  • Средняя задержка (мс): прогноз R для целевого RPS.
  • Максимальная (мс): R_max ≈ R_avg × (max/avg) базовой ступени.
